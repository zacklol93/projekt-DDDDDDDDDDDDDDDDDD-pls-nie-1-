- Ustawienie konfiguracji, która działa dla Ciebie.  
- Kontrola odrzutu i celowanie.  
- Poruszanie się i pozycjonowanie. :contentReference[oaicite:1]{index=1}

###  Komunikacja i strategia
- Efektywna komunikacja z drużyną.
